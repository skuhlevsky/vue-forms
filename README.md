# vue-forms

[Online Example](https://vue-forms.stas.tv/)

## Описание проекта

Констуктор форм принимает данные для построение формы.
Пример:
userData: {
name: 'Vik',
email: 'to@vik.site',
gender: 'male',
sub: '1980-12-10',
},

Проверка полей осуществляется по имени свойства validator конструктора полей формы.
